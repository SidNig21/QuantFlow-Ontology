## Objective

Prove the foundation the founder actually needs next: **two Hermes sessions on the canvas at once**, and a **Kernel-mediated handoff** from one to the other — not a guest-to-guest side channel, not “datasets,” not another fake chat UI.

Acceptance is a runnable demo: Spawn Hermes twice → two term tiles → orchestrator path issues a Kernel-backed delegate/message → worker Hermes receives it → both Kernel rows stay honest; orphans zero.

WO-008d put one Hermes TUI on the desk. The end goal is Hermes as **orchestrator of other agents/processes on the canvas**. That is unproved until two Hermes tiles collaborate through QuantFlow. Datasets (WO-009) stay parked until this seam holds.

## Deliverables

**0. Measure the gap (paste unedited).**
- Can dock Spawn Hermes twice today → two live Kernel sessions + two term tiles? (Expect yes; paste session ids.)

**1. Two Hermes tiles are first-class.**
Spawn (or host admit) **two** `native_tui` Hermes sessions. Canvas shows **two** term tiles. Both Kernel rows `created`/`started` (or current legal pair). Cancel/close either → that PTY dies; the other stays; orphan check OK for both.

**2. Kernel-mediated handoff A → B.**
Implement the smallest host path such that:
- Session A (orchestrator) triggers a **Kernel command/event** naming B (and a payload / task text).
- The host delivers that payload into B (PTY inject and/or host ACP prompt — measure which is viable with two TUI sessions; document the choice in one sentence).
- B’s tile visibly reacts (text appears / turn starts). A did **not** open a private socket/file pipe to B.
- Falsify: with the Kernel/host delivery path disabled or broken, B does not receive the message (red); restore → green. Paste both outputs.

**3. No guest-to-guest side channel.**
Gate or smoke that the handoff goes through Kernel (+ host adapter). Direct Hermes↔Hermes networking or shared scratch files as the control plane = fail. Agent-private Hermes memory may exist; it is not the QuantFlow receipt.

**4. Hygiene.**
Extract-first on shell JS (`renderer.js` / `tile-manager.js` must not grow). `agent-host.ts` under 1k. Static gates green; `cd collab-electron && bun run build` exit 0. Evidence under `docs/orders/evidence/wo-008e/`.
