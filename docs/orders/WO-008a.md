## Objective

Wire ACP **permission requests** on the host bridge (deny-by-default, founder-visible grant) and unlock `runTurn` for `host_acp` sessions so the founder can complete a **real Hermes turn** from the dock/session tile — without reviving legacy auto-approve (`acp-agent.ts`, debt #14).

## Why this order exists (measured)

WO-008c landed host `hermes acp` admit (Outcome A) + Kernel `created`/`started`. `requestPermission` already **denies**; `runTurn` **throws** on `host_acp` until this order. Hermes will request approvals on tool use ([Hermes ACP](https://hermes-agent.nousresearch.com/docs/user-guide/features/acp) — allow once / always / deny; timeout → deny). Without a bridge, a live turn is either stuck or silently cancelled.

**0. Permission bridge (host).**
Replace handshake-only deny with a real ACP permission path: pause for founder decision on dangerous calls; map outcomes to Hermes options (`allow_once` / `allow_always` / deny). Default remains **deny** on timeout / missing UI / cancel. Do **not** copy `acp-agent.ts` auto-approve. Prefer extending `host-acp-client.ts` (or a thin sibling) so smokes and Electron share one client.

**1. `runTurn` for `host_acp`.**
Lift the WO-008c forbid. Prompt → stream chunks into the session tile (same IPC/events shape as AgentOS turns where possible). Cancel mid-turn tears down cleanly (no orphan hermes pids — reuse WO-008c orphan check).

**2. Per-species tool policy.**
Enforce an allowlist at the host seam (species data / `agent_definition` — **stop and report** if a schema column is truly required). Falsify: unlisted tool via critic-mock or a test species → denied; listed harmless tool → may proceed under the permission bridge rules.

**3. Founder live path (documented, not a secret).**
Register Hermes → dock Spawn → **Run turn** with a short harmless prompt the founder chooses. Paste: session id, that chunks arrived, permission UI evidence (grant or deny once). No credentials in logs.

**4. Gate hygiene (carry-forward from WO-008c verify).**
`kernel-sole-writer-app` allowlists `host-acp-bridge.ts` but the SDK import lives in `species/hermes/host-acp-client.ts` (outside the gate walk). Either: allowlist + scan that path, **or** move the shared client under a package/`collab-electron` home the gate already covers. Neuter → red → restore → green.
