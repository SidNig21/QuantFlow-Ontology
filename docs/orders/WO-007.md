**3. The host resolves species from rows, not code.** `agent-host.ts`'s in-code species map shrinks to nothing or to pure path-resolution plumbing: spawn takes a species *name*, loads the row, launches `package_ref`. **SDK path (measured, `@rivet-dev/agentos-core` `agent-os.d.ts:554`):** packages for rows known at host creation go into `AgentOs.create`'s `software` list; a row registered **after** the host exists is admitted via `await os.linkSoftware(...)` before `createSession(name)` ‚Äî no host re-create. `package_ref` is a path resolved from the app root (absolute allowed); resolution failure is the typed error.
