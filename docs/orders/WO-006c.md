## Objective

From the running app: spawn an agent session, watch it stream into a tile, see it call one harmless tool, and end with a durable Artifact published through the Kernel — with cancel, fail, retry, and close all real, and a force-kill mid-run leaving honest terminal state instead of phantom sessions.
